Overview

I was approached to create a simpler, more stylized website for the Swerve's Bar fan podcast. This website redesign changed the back-end infrastructure of the website to be simpler, and easier to update (e.g. using normal text markup for each episode instead of JSON objects). The organization of the website, including text and page layout, was improved to make it easier for new listeners to find relevant information. Aesthetically, we moved from default ASP.NET styling to a futuristic, vaporwave, late-night cafe feel to fit the podcasts' namesake (a bar referenced in the larger Transformers universe).
